Stagetime

Stagetime is a community blogging platform with a full feature set including a sorted feed, a text editor, authorisation and commenting, the ability to save articles to your reading list and edit your bio.

It's an isomorphic, progressive web application with an emphasis on performance and SEO. The build was done using React (with NextJs), Node (with Express) and MongoDB. I utilized React libraries for infinite scrolling, auth and the text editor.

Notflix

NOTFLIX is a cheeky clone of a well known entertainment provider. The app calls movies from a API and displays them in a pleasing and responsive way. There's some interactive functionality, including the ability to add movies to your watch list and to search the movie database.

The build was done using React (w/Hooks).

Newscycle

Newscycle is a news aggregation app that pulls in a variety of international stories and features.

The app is built in React and uses newsapi.org as its source. Future iterations of the site will include section navigation and personalisation.

About me

Over a ten year stint in journalism, working mostly at the Guardian, I slowly moved from writing to increasingly technical roles, ultimately working directly with developers on projects to benefit the newsroom.

Proximity to the dev teams inspired me to reignite my interest in coding and I began teaching myself the fundamentals of web development, picking up skills over a period of almost two years before finally consolidating and expanding my skills at the Codeworks bootcamp.

Technical knowledge
